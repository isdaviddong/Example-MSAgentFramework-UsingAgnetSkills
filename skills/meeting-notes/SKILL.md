## Instructions

When processing a meeting transcript:

1. **Key Discussion Points** — Extract the main topics discussed during the meeting.
2. **Decisions Made** — List any decisions that were agreed upon by participants.
3. **Action Items** — Create a clear list of action items, each with:
   - Task description
   - Owner (who is responsible)
   - Due date (if mentioned)
4. **Summary** — Keep the overall summary concise (3-5 sentences max).

Format the output using markdown with clear headings for each section.
